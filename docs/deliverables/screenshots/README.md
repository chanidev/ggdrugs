## 캡처용 데모 계정 (개발 DB)

- **user** = `테스터` (채팅방·약속·메이트·크레딧·알림·북마크·리뷰·커뮤니티 글 보유)
- **uploader** = `데모업로더` (승인된 업로더 프로필 + 등록 이벤트 2건: 승인/대기)
- **admin** = `데모관리자` (admin_profile 보유)

데모 데이터는 `seed-chat-demo.ts` + 캡처용 보강 시드로 생성. 로그인은 개발 전용 `dev-login`(닉네임) 사용.
