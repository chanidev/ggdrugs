# 메인 페이지 플로우

## Summary

서울시 자치구 지도를 기반으로 한 이벤트 탐색 진입점. 로그인 전/후 상태 분기가 있고, 클러스터 마커 클릭 → 지역 이벤트 목록, 목록/지도 뷰 토글, 필터 해제 동작을 포함한다. 일반 사용자의 1차 접점이며, 검색·필터 5종(지역/기간/인원구성/이벤트 종류/이벤트 성향) 적용 전 기본 상태가 이 화면이다.

## Key points

- **지도 배경**: 서울시 자치구 경계 + 클러스터 마커 (Kakao Maps 추정).
- **로그인 상태**:
  - 비로그인: 탐색만 가능, 북마크·리뷰 버튼 비활성.
  - 로그인: 사용자 정보 헤더, 북마크 토글 활성.
- **클러스터 클릭**: 해당 자치구의 이벤트 결과 목록이 사이드/하단 패널로 노출.
- **뷰 토글**: 목록 ↔ 지도 전환. 동일 필터 조건이 양 뷰에 유지된다.
- **필터 해제("취소하기")**: 적용된 필터를 초기 상태로 되돌리고 메인 지도 뷰로 복귀.

## Open questions / contradictions

- ~~"취소하기"의 정확한 의미 확정 필요~~ → **해소**: 요구사항정의서 GG-FILTER-003 "필터 검색 취소" = 필터 초기화로 확인됨.
- 서울 외 지역 확장 시점 미정 (현 초안은 서울시 자치구만 표시).
- 클러스터 결과 목록의 정렬 기준 미정 (거리순 / 인기순 / 최신순).
- 로그인 유도 지점(CTA) 위치 미확정.
